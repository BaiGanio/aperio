How Aperio works on your computer

What the installer added, where your data lives, and how to change things — no technical knowledge needed.

What was installed, and why

Aperio set up everything it needs by itself. Nothing was installed system-wide with administrator rights.

ComponentWhat it isWhere it lives
Node.js The engine Aperio's app runs on. Installed only for your user account (via nvm), so it never touches the rest of the system. ~/.nvm
Ollama The program that runs the AI model itself, fully offline. Aperio keeps its own private copy so it never interferes with anything else on your machine. Aperio folder → vendor/ollama
An AI model The "brain" — chosen during setup to fit your computer's memory. It is a large download (several GB), stored by Ollama. Ollama's model storage
App dependencies The libraries the Aperio app is built from. Aperio folder → node_modules/

Where your data lives

Everything you create stays inside the Aperio folder — nothing is sent anywhere:

  • Your memories, conversations and settings — a database file at .sqlite/aperio.db
  • Files you attach, logs, sessions — under var/

Back up the Aperio folder and you have backed up everything.

Starting and stopping

  • Day to day: double-click the Aperio icon on your Desktop. Your browser opens by itself — no terminal window.
  • First run only: a terminal window stays open. That window is Aperio's engine — keep it open while you use the app.
  • Stopping: Aperio stops itself a few minutes after you close the last browser tab. You can also press the Quit button (power icon) in the app's top bar.
  • If the browser doesn't open by itself, go to http://localhost:31337.

Changing settings after installation

Everything is changed from inside the app — you never need to edit files:

  • AI model — the model menu in the top bar. Trying different models is encouraged: they have different speeds, skills and personalities.
  • Language — the flag menu in the top bar (all 24 official EU languages).
  • Theme, voice, and everything else — the Settings panel (gear icon).
  • Cloud AI (optional) — if you have an API key from a cloud provider, you can switch to it in Settings. Local stays the default; nothing requires a key.

Working with your documents

This is where Aperio shines for everyday work. It understands PDF, Word, Excel, PowerPoint, plain text, Markdown, email files, and images (photos and scans are read by a local vision model). Try:

  • Attach a PDF and ask "what is this document about?"
  • Attach a spreadsheet and ask questions about the numbers in it.
  • Ask Aperio to write a Word document or build an Excel file for you.
  • Attach a photo of a paper document and ask what it says.
Whole folders of documents are searchable at once: the document index is on out of the box, watching your allowed folders. Put files there, then ask a question — Aperio finds the right passage across all of them.

Uninstalling

  • macOS / Linux: open the Aperio folder and run uninstall.sh. It removes what Aperio installed (its Ollama copy, dependencies, data) and offers to delete the downloaded AI model. It never touches software you already had.
  • Windows: delete the Aperio folder, then remove the model with Ollama if you wish. A one-click uninstaller is on its way.
  • Finally, drag the Aperio folder to the Trash — everything else lives inside it.

Something not working?

  • App won't load → make sure Aperio is running (Desktop icon), then visit http://localhost:31337.
  • Answers are slow → try a smaller model from the model menu; smaller models are much faster on modest hardware.
  • Setup problems → technical details are written to var/install/ inside the Aperio folder.
