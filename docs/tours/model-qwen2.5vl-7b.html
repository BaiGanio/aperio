Qwen2.5VL 7B tools + vision

A 7B model that happens to have eyes. "VL" means Vision-Language: show it a photo, a screenshot, a diagram, a handwritten note — and it tells you what it sees. Its text skills match a typical 7–8B model; the reason you choose it is the images.

Hardware & setup

As a single model, its needs match Llama 3.1 8B. Image processing adds a little overhead, so descriptions come back slightly slower than plain chat.

  • Apple Silicon · 16 GB+Fits in GPU-accessible memory with headroom. On 32 GB you can run it alongside a stronger chat model (see below).
  • PC · NVIDIA GPU, 8 GB+ VRAMAn RTX 2070 / 3060 or better. A second model needs extra VRAM or system RAM.
  • PC · CPU onlyWorks at ~5–10 words/sec; expect a touch slower when describing images.
Two ways to run it. Either as your only model (it both chats and sees), or as a dedicated vision model beside a stronger chat model — Aperio routes image tasks to the VL model automatically. The second needs enough RAM for both (32 GB+). In .env:
LLAMACPP_CTX=32768 LLAMACPP_MODEL=ggml-org/gemma-4-12B-it-GGUF:Q4_K_M # your preferred chat model LLAMACPP_VLM_MODEL=ggml-org/Qwen2.5-VL-7B-Instruct-GGUF # used only for images

Try it yourself

You'll need a few images on your computer — phone photos, screenshots, a picture of a handwritten note. Upload one with the image button (or type its file path) and run these.

1 · Describe an image — the main event

Pick a photo with a few distinct elements: a room, a street scene, objects on a table.

Paste this (with an image attached)Describe this image in detail. What do you see? Be specific about objects, colours, layout, and anything that stands out.
✅ GoodA concrete description that feels like someone narrating over your shoulder — "a red ceramic mug to the left of a silver laptop" — not vague "some objects on a surface". It picks up details you might have missed.
❌ The wallA vague, generic description. Usually means a blurry or dark image — try a clearer, well-lit one with recognisable objects.
→ MeansYour AI can see. Every other model in the lineup is text-only; this one isn't.

2 · Read text from a photo, then use it

Photograph a document, sign, menu, or book page — then ask it to read and act on the words. This chains sight + comprehension.

Paste this (with a text image attached)Read all the text in this image and transcribe it exactly. Then summarise the main point in one sentence.
✅ GoodNear-perfect transcription of printed text (good on neat handwriting too), followed by a correct one-sentence summary. Photograph a document, get back what it says.
❌ The wallGarbled text from messy handwriting or a low-resolution photo. Printed text and tidy writing work far better than a doctor's scrawl.
→ MeansA genuinely useful everyday workflow — turn any photographed page into editable, summarised text.

3 · Compare two images

Attach two pictures — two rooms, two products, two designs. Aperio supports multiple images in one message.

Paste this (with two images attached)Compare these two images. What's similar, what's different, and which one feels more inviting — and why?
✅ GoodIt holds both in view and reasons across them: "Both have natural light from the left, but the first uses warm wood tones while the second is white and grey — the first feels cosier."
❌ The wallIt describes only one, or makes a comparison that doesn't match the pictures.
→ MeansAn objective second eye for design choices, shopping comparisons, and before-and-afters.

4 · Vision + memory + wiki, together

Combines sight with tool calling — the model seeing your space, remembering it, and writing it up.

Paste this (with a photo of your desk/room)Here's a photo of my workspace. Please: (1) describe it in detail; (2) remember this as my current setup; (3) write a wiki article called "My Workspace" with the description and three layout improvements based on what you see.
✅ GoodStrong description, then it calls remember and wiki_write. The vision part is reliable; the tool part is ~85% like Llama 3.1 8B — a retry is occasionally needed.
❌ The wallIt describes the photo but skips a tool step. Retry naming the tool, or pair it with a stronger chat model (above).
→ MeansIt can act on what it sees — the suggestions, grounded in your actual space, are the payoff.

The verdict

Qwen2.5VL 7B isn't the strongest text model or the most reliable tool user — but it does something none of the others can: it sees. Choose it if you regularly work with photos, screenshots, or documents. For the best of both worlds, pair it as a dedicated vision model alongside Gemma 4 12B or Qwen3 30B-A3B (needs 32 GB+).
