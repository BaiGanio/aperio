Model tours

A deep, hands-on look at every model you can run in Aperio — what each one is genuinely good at, the hardware it needs, and tasks scaled to its real ability.

The Model Guide gives you the quick overview and a side-by-side comparison table. These tours go the other way — deep on one model at a time. Each picks up where the table leaves off: the hardware it really needs, what it can and can't do, and concrete tasks tuned to that model's level so you're never fighting a wall it was never going to clear.

Read them smallest to largest to feel the capability ladder, or jump straight to whichever model you've set up.
