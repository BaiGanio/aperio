Evaluate Your Model

Model evaluation suite

Hands-on, copy-paste test suites that probe what an AI model in Aperio can really do — not what its spec sheet claims. No technical knowledge needed: pick a suite, paste the prompts, mark pass or fail.

The Model Guide gives you a quick five-test calibration to learn any model's broad shape. These suites go deeper. Each one targets a single capability that's easy to claim and hard to verify — and shows you exactly what passing and failing look like, so you can trust your own eyes instead of the marketing.

Every suite has a built-in scorecard you can fill in as you go and print when you're done. Run a suite on each model you've set up, and you'll know which model to point at which job.
