Exam Sections

  • §0 Fixture Import — 28 memories tagged aperio-exam. Required before any drill.
  • §1 Memory Tools — 10 drills: exact recall, semantic recall, preference, forget, update, list, tag filter, time filter, temporal, batch.
  • §2 Wiki — 5 drills: write, read, search, list, refresh.
  • §3 Code Graph — 7 drills: symbol search, source slice, call graph, definition jump, reference search, file outline, cross-file trace.
  • §4 File Tools — 9 drills: read text, read PDF, read image, write, edit, patch, directory listing, file search, grep.
  • §5 Shell — 6 drills: basic command, pipe, redirect, exit code, timeout, sandbox rejection.
  • §6 Web — 3 drills: fetch URL, web search, GitHub issue fetch.
  • §7 Skills Matched — 17 drills: each canonical skill prompt fires the right skill.
  • §8 Multi-Tool Chains — 4 drills: search→read→edit, fetch→write, recall→update, code-graph→edit.
  • §9 Guardrails — 4 drills: secret file denial, SSRF block, shell sandbox, content fencing.
  • §11 Roundtable — multi-agent discussion. Requires ≥2 models configured. Not scored.

Before You Start

Prerequisites:
  • Shell: Set APERIO_ENABLE_SHELL=1 for §5 shell drills
  • Roundtable: Configure ROUNDTABLE_AGENTS with ≥2 provider:model pairs for §11
  • Wiki refresh: Set WIKI_REFRESH_PROVIDER for §2.5
  • GitHub token: Set GITHUB_TOKEN for §6.2 to avoid rate limits
  • Code graph: Index the repo first — use the Code panel or ask the agent
  • Embedding backfill: After fixture import (§0), wait ~10s before semantic recall drills in §1.2
