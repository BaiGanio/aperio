⟐ E2E Tests

End-to-end verification suite — WebSocket lifecycle, streaming protocol, config provenance, settings round-trip, and UI config panel.

How to read this page: each green dot means one behavior was checked successfully. A test is a small question about the running application; it does not mean a person had to click through the interface.

The suites group related questions. Expand a suite to see both the technical check name and a plain-language explanation of what it protects. “Mocked” means the test uses safe stand-ins instead of a real database, filesystem, or external service.
