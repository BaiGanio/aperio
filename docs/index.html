Features

Everything you need.
Nothing you don't.

Built for developers who want AI that actually knows them — without giving up their data or their stack.

Persistent Memory

Memories survive every conversation, every tool, every session. 7 structured types keep things organized — facts, decisions, solutions and more.

Semantic Search

Powered by pgvector + Voyage AI. Ask for "my TypeScript projects" and get results by meaning, not just keyword matching.

MCP Compatible

One brain shared across every agent. Claude, Cursor, Windsurf — they all connect to the same Postgres database via the MCP protocol.

Real-time Streaming

Responses stream token by token. Blinking cursor, live code rendering, markdown processed on completion. No waiting.

Cloud & Local AI

Switch between Claude (cloud) and Ollama (local) with one env var. No code changes. Same memory, same tools, different provider.

Auto Deduplication

Background job every 10 minutes finds near-duplicate memories using cosine similarity. Dry-run by default — you stay in control.

Architecture

Simple stack.
Serious power.

Quick Start

Up in 5 minutes.

Node 18+, Docker, an Anthropic key and a Voyage AI key (free tier). That's all.

01
Clone & install
# clone the repo
git clone github.com/
BaiGanio/aperio
cd aperio && npm i
02
Configure .env
cp .env.example .env
 
# add your keys
ANTHROPIC_API_KEY=...
VOYAGE_API_KEY=...
03
Start database
cd docker
docker compose up -d
cd ..
# run migrations
docker exec -i ...
04
Launch Aperio
# cloud (Claude)
npm run start:cloud
 
# local (Ollama)
npm run start:local
