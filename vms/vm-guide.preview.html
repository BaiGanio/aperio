When something is red

The short troubleshooting map.

01
Native module fails

The guest probably received the wrong architecture’s node_modules. Remove guest dependencies and run npm install inside the guest.

02
Guest never becomes ready

Parallels Tools may be missing, or the shared-folder path may be wrong. Reboot the guest and confirm the share is visible.

03
Docker image missing

Local tags are never pulled automatically. Build aperio:test-local first, then run the Docker command again.

Keep the evidence: runner logs are written under vms/out/. They are disposable runtime artifacts—inspect them when debugging, never commit them.
